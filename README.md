# Company Profile Website

A company profile website for DevForge. This project was built as part of our Week 1 training.

## Folder Structure

```
Company-Profile-Website/
├── Home.html
├── About.html
├── Services.html
├── Portfolio.html
├── Contact.html
├── index.html

├── Home.json
├── Aboutus.json
├── Services.json
├── Portfolio.json
├── Contact.json
│
├── API Doc.md
├── Database Design.md
├── README.md
```

## JSON Files

Each JSON file stores the data for one page of the website.

- **Home.json** — Company info, capabilities, features, technologies, featured projects, and testimonials for the Home page.
- **Aboutus.json** — Company story, core values, team members, and development process for the About page.
- **Services.json** — Services list, how-we-work steps, and FAQ for the Services page.
- **Portfolio.json** — Project list for the Portfolio page.
- **Contact.json** — Company contact info for the Contact page.

## Documentation Files

- **API Doc.md** — Lists the API endpoints, their methods, and JSON examples.
- **Database Design.md** — Describes the entities and their relationships at a conceptual level.
